NyandSign — Chrome Web Store 商品説明（日本語）


短い説明（132文字以内）

ウェブカメラでハンドサインを検出し、メディア再生やスクロール・タブ切替などのブラウザ操作をハンズフリーで実行。


詳細な説明

NyandSign とは？

NyandSign は、ウェブカメラでハンドサインを検出し、ブラウザ上のメディア操作やブラウザ操作をハンズフリーで行う Chrome 拡張機能です。Google MediaPipe のオンデバイス AI を使い、手の動きをリアルタイムに認識します。すべての処理はブラウザ内でローカルに完結し、カメラ映像が外部サーバーに送信されることはありません。

YouTube、Spotify、Netflix などのメディア再生はもちろん、ページスクロール、戻る/進む、タブ切り替え、ズームなどもハンドサインで操作できます。通常のウェブカメラに加えて、対応する電気メガネのカメラも利用できます。

📌 PiP（ピクチャーインピクチャー）モード
フルスクリーン再生時にサイドパネルが邪魔になる場合、コンパクトなポップアップウィンドウに切り替え可能。PiP ではメディア操作モードは緑枠、ブラウザ操作モードは青枠で表示され、現在のモードを把握しやすくしています。

🎯 操作対象タブの指定・固定
ジェスチャー操作を特定のタブに送信できます。他のページを閲覧しながら、BGMを再生しているタブや操作したいタブを固定できます。

🔒 完全ローカル処理・プライバシー保護
すべてのハンドトラッキングは WebAssembly を使用してデバイス上で処理されます。カメラ映像がブラウザの外に出ることはありません。クラウド処理なし、データ収集なし、アカウント登録不要です。

🌐 日英バイリンガル対応
日本語・英語のインターフェースに対応しています。


NyandSign をインストールするメリット

ハンズフリーの利便性 — 料理中、運動中、食事中など手が塞がっている時でもメディアやページを操作できます。

アクセシビリティ — キーボードやマウスの操作が困難なユーザーに代替入力手段を提供します。

電気メガネ連携 — XREAL などの対応電気メガネに搭載されたカメラでハンドサイン操作が可能。メガネをかけたまま直感的に操作できます。

視聴体験を損なわない — PiP モードによりフルスクリーンコンテンツの邪魔をせず、ジェスチャー操作を維持できます。

自分の使い方に合わせられる — メディア操作とブラウザ操作を別々に設定でき、よく使うサインに必要な操作を割り当てられます。

追加ソフト不要 — 軽量な Chrome 拡張機能だけで完結。デスクトップアプリ、ドライバー、アカウント登録は不要です。


権限が必要な理由

NyandSign は機能に必要な最小限の権限のみをリクエストします。各権限の用途と必要な理由を詳しく説明します。

activeTab（アクティブタブ）
拡張機能アイコンをクリックした際に、現在のタブと対話する権限です。表示中のページにコンテンツスクリプトを注入し、メディア操作やページスクロールなどのコマンドを伝えるために必要です。

tabs（タブ情報）
タブのタイトルを読み取り、操作対象タブの表示・固定や、次のタブ / 前のタブへの切り替えに使用します。タブの情報はサイドパネル内でのローカル表示と操作対象の判定にのみ使用され、外部に送信されることはありません。

scripting（スクリプト注入）
ウェブページに小さなコンテンツスクリプトを注入する権限です。このスクリプトは拡張機能とページ上のメディア要素・スクロール操作をつなぐ橋渡し役です。ジェスチャーは検出できても、この権限がないとページ上の再生操作やスクロール操作を実行できません。

storage（ストレージ）
ハンドサインの割り当て、操作モード、音量設定、表示オプションなどを Chrome の組み込みストレージに保存する権限です。ブラウザを再起動しても設定が維持されます。Chrome 同期を有効にしている場合は Chrome により端末間同期されることがありますが、NyandSign 自身が外部サーバーへ送信することはありません。

sidePanel（サイドパネル）
Chrome のサイドパネルとして拡張機能のインターフェースを表示する権限です。サイドパネルはカメラ映像、ハンドトラッキングの可視化、認識中のサイン、操作モード、各種設定、チュートリアルを表示するメインUIです。

ホスト権限（全URLへのアクセス）
メディアやスクロール対象はあらゆるウェブサイトに存在するため、コンテンツスクリプトは全ページで動作する必要があります。YouTube、Spotify、Netflix、Twitch、SoundCloud、ニュースサイト、ドキュメントサイトなど幅広いページで操作するためです。コンテンツスクリプトはジェスチャーコマンドの実行に必要なメディア要素・スクロール操作のみを扱い、カメラ映像や個人情報を外部に送信しません。
